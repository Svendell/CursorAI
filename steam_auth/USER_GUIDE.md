STEAM AUTH MANAGER - РУКОВОДСТВО ПОЛЬЗОВАТЕЛЯ
==============================================

## ГЛАВНЫЙ ЭКРАН (Home)

После запуска приложения вы видите главный экран с четырьмя основными кнопками:

```
┌─────────────────────────────────┐
│  Steam Auth Manager             │
│  Manage your Steam 2FA          │
│                                 │
│ [Manage Accounts (0)]           │ ← показывает кол-во аккаунтов
│                                 │
│ [Add New Account]               │ ← добавить новый аккаунт
│                                 │
│ [Import mafile]                 │ ← импортировать существующий
└─────────────────────────────────┘
```


## ДОБАВЛЕНИЕ НОВОГО АККАУНТА

1. Нажимите "Add New Account"
2. Заполните обязательные поля:
   - **Username** - ваше имя пользователя Steam
   - **Password** - пароль от аккаунта (минимум 3 символа)
3. Опционально:
   - **Phone Number** - номер телефона связанный со Steam
   - **Steam ID** - ID аккаунта (если известен)
   - **Shared Secret** - секрет для 2FA (если уже есть)
4. Нажимите "Add Account"
5. Если все корректно - увидите сообщение об успехе (зеленое)
6. Автоматически вернетесь на экран списка аккаунтов


## СПИСОК АККАУНТОВ

Все добавленные аккаунты отображаются в виде списка:

```
┌─────────────────────────────────┐
│  Accounts                       │
│                                 │
│ ┌─────────────────────────────┐ │
│ │ username1                   │ │ ← тап для открытия
│ ├─────────────────────────────┤ │
│ │ username2                   │ │
│ ├─────────────────────────────┤ │
│ │ username3                   │ │
```

- Нажмите на аккаунт чтобы открыть его информацию
- Нажмите "Add Account" чтобы добавить еще один
- Нажмите "Back" чтобы вернуться на главный экран


## ИНФОРМАЦИЯ ОБ АККАУНТЕ

При клике на аккаунт откроется страница с информацией:

```
┌─────────────────────────────────┐
│ username1                       │
│ 2FA Code: 23456                 │ ← обновляется каждые 30 сек
│                                 │
│ ┌─────────────────────────────┐ │
│ │ Confirmations │ Export       │ │
│ ├────────────────┼──────────────┤ │
│ │ Copy Code     │ Delete       │ │
│ └─────────────────────────────┘ │
│                                 │
│ Steam ID: 76561198765...        │
│ Created: 2026-01-15             │
│                                 │
│ [Back]                          │
└─────────────────────────────────┘
```

### Действия на этом экране:

**Confirmations** - открыть операции для подтверждения на Steam (торговля, продажа на маркете)

**Copy Code** - скопировать 2FA код в буфер обмена

**Export** - экспортировать аккаунт в mafile (можно использовать на другом устройстве)

**Delete** - удалить аккаунт из приложения (может потребовать подтверждение)


## ИМПОРТ MAFILE

1. Нажимите "Import mafile" на главном экране
2. В FileChooser выберите файл (.maFile или .json)
3. Если файл зашифрован - введите пароль
4. Нажимите "Import"
5. Если всё ОК - появится сообщение об успехе
6. Аккаунт будет добавлен в приложение


## ЭКСПОРТ АККАУНТА

1. Откройте информацию об аккаунте
2. Нажимите "Export"
3. Выберите аккаунт из списка (если их несколько)
4. Опционально введите пароль для защиты файла
5. Нажимите "Export"
6. Файл будет сохранен в формате .maFile


## ОПЕРАЦИИ ДЛЯ ПОДТВЕРЖДЕНИЯ (Confirmations)

Если на вашем Steam аккаунте есть незавершенные операции:

1. Откройте аккаунт
2. Нажимите "Confirmations"
3. Список показывает ожидающие операции:
   - Торговля предметами
   - Продажа на маркете
   - И другие операции требующие подтверждения
4. Нажмите "Accept" или "Decline" для каждой операции


## РЕДАКТИРОВАНИЕ АККАУНТА

1. Откройте аккаунт
2. Нажмите "Edit" (если доступно)
3. Измените имя аккаунта если необходимо
4. Нажмите "Save" для сохранения или "Cancel" для отмены


## УДАЛЕНИЕ АККАУНТА

1. Откройте аккаунт
2. Нажмите "Delete"
3. Подтвердите удаление во всплывающем окне
4. Аккаунт будет удален из приложения


## ВАЛИДАЦИЯ ВХОДНЫХ ДАННЫХ

Приложение проверяет:
✓ Username не может быть пустым
✓ Password обязателен и минимум 3 символа
✓ Phone Number - корректный формат (опционально)
✓ Steam ID - только цифры (опционально)
✓ Shared Secret - правильный base64 (опционально)

Если данные некорректны - увидите сообщение об ошибке (красное)


## СООБЩЕНИЯ О СТАТУСЕ

Разные цвета обозначают:

🟢 ЗЕЛЕНОЕ - операция успешна завершена
🔴 КРАСНОЕ - произошла ошибка
🟡 ЖЕЛТОЕ - операция в процессе


## ЛОГИРОВАНИЕ

Все операции логируются в файл logs.txt:
- Добавление аккаунта
- Удаление аккаунта
- Импорт/экспорт маfiles
- Все ошибки

Для просмотра логов:
```bash
cat logs.txt
```


## СТРУКТУРА MAFILES

Маfiles хранятся в директории `mafiles/`:
```
mafiles/
├── 765611234567890.maFile
├── 765611987654321.maFile
└── manifest.json
```

manifest.json содержит список всех аккаунтов и их параметров.

## РЕШЕНИЕ ПРОБЛЕМ

**Проблема**: Ошибка при добавлении аккаунта
**Решение**: Проверьте что введены все обязательные поля

**Проблема**: Не могу импортировать маfile
**Решение**: Убедитесь что файл существует и имеет правильное расширение (.maFile или .json)

**Проблема**: 2FA код не обновляется
**Решение**: Экран обновляется каждые 30 секунд. Проверьте что Shared Secret корректен

**Проблема**: Ошибка при экспорте
**Решение**: Убедитесь что aккаунт содержит все необходимые данные
