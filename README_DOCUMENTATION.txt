📊 КЛЮЧЕВАЯ ИНФОРМАЦИЯ:

Минимальная структура mafile для 2FA:
{
  "shared_secret": "FhkMQfG2w3Z9nBvK7xL2mN4pQ6rS8tU1wV2xY3zA=",
  "account_name": "mysteamaccount"
}

TOTP Алгоритм (генерирование 2FA кода):
1. Декодировать shared_secret из Base64 → 20 байт
2. time_counter = current_time // 30 (30-сек периоды)
3. time_bytes = big_endian_8_bytes(time_counter)
4. hmac_hash = HMAC-SHA1(secret_bytes, time_bytes)
5. index = hmac_hash[-1] & 0x0f
6. code = (hmac_hash[index:index+4] % 100000).zfill(5)

Результат: 5-значный код (например: "12345")

🔐 КРИТИЧНЫЕ МОМЕНТЫ БЕЗОПАСНОСТИ:

❌ НИКОГДА не делайте:
- Логируйте полный secret
- Сохраняйте secrets в plaintext
- Передавайте secrets в URL параметрах
- Копируйте secrets в памяти

✅ ВСЕГДА делайте:
- Шифруйте secrets перед сохранением
- Логируйте только с маской: secret[:8]+"..."
- Передавайте через POST + HTTPS
- Очищайте память после использования
